# Level 1 — Beginner: Building Confidence with Excel

---

## Course Overview

| Detail | Information |
|---|---|
| **Level** | Beginner |
| **Duration** | 4 weeks |
| **Sessions** | 2 sessions per week × 1.5 hours per session |
| **Mode** | Live virtual classes with practice exercises and optional Q&A |
| **Prerequisites** | None — open to all women aged 18+ |

**Goal:** Build foundational Excel skills, basic data understanding, and confidence using data for simple analysis and visualization.

---

## Learning Objectives

By the end of Level 1, participants will be able to:

- Navigate Excel confidently and manage workbooks, worksheets, and cell references
- Clean and organize raw data for basic analysis and interpretation
- Use foundational formulas and functions accurately
- Create and format tables and charts for simple data stories
- Apply practical Excel skills to everyday or community-related data problems

---

## Weekly Breakdown

---

### Week 1 — Getting Comfortable with Excel
**Theme:** *From Zero to Familiar*

**Session 1: Introduction to Excel and the World of Data**
- What data analysis means in real life
- Excel interface: ribbons, cells, ranges, and keyboard shortcuts
- Workbook vs. worksheet navigation
- Basic formatting: fonts, borders, colors, and alignment
- Saving, naming, and organizing files

*Hands-On Activity:* Organize a simple contact list or class attendance sheet
*Mini-Assignment:* Create a neat table with your weekly schedule or personal goals

---

**Session 2: Data Entry, Sorting, and Basic Organization**
- How to enter and edit data correctly
- Understanding data types — text, numbers, and dates
- Sorting and filtering basics
- Data cleaning: removing blanks and correcting errors
- Introduction to the "Format as Table" feature

*Hands-On Activity:* Clean a small dataset of women entrepreneurs and their monthly sales
*Mini-Assignment:* Create a formatted, sorted table showing name, location, and monthly income

---

### Week 2 — Formulas and Functions Made Simple
**Theme:** *Doing the Math with Excel*

**Session 3: Basic Formulas and Calculations**
- How formulas work (=, +, -, *, /)
- Introduction to functions: SUM, AVERAGE, MIN, MAX
- Using AutoFill and basic formula troubleshooting
- Understanding cell references: absolute vs. relative

*Hands-On Activity:* Create a simple budget tracker to calculate total expenses and averages
*Mini-Assignment:* Build a basic income vs. expense sheet using formulas

---

**Session 4: Logical Thinking in Excel**
- Understanding and using IF statements
- Combining conditions with simple logic
- Common errors (#VALUE!, #DIV/0!) and how to fix them

*Hands-On Activity:* Create a performance tracker using IF formulas (e.g., student grades or sales results)
*Mini-Assignment:* Create a sheet that categorizes data using Pass/Fail or High/Low outcomes

---

### Week 3 — From Raw Data to Visual Insights
**Theme:** *Telling Stories with Data*

**Session 5: Visualizing Data with Charts**
- When and why to use charts
- Creating bar, column, and pie charts
- Adding titles, labels, and colors for clarity
- Chart best practices — avoiding visual clutter

*Hands-On Activity:* Create a chart showing top-performing products or survey responses
*Mini-Assignment:* Present your data in two chart types and explain which communicates better

---

**Session 6: Data Summarization and Formatting for Reports**
- Using AutoSum, Quick Analysis, and basic descriptive summaries
- Applying conditional formatting to highlight insights
- Adding headers, footers, and basic page setup for print or sharing

*Hands-On Activity:* Format an attendance dataset and highlight key trends with color scales
*Mini-Assignment:* Prepare a short "Data Summary Sheet" using a class-provided dataset

---

### Week 4 — Practice, Confidence, and Capstone Project
**Theme:** *I Can Analyze Data!*

**Session 7: Guided Practice and Real-World Applications**
- Recap of key concepts from Weeks 1–3
- Group challenge: cleaning and analyzing a shared dataset
- Discussion: real-life Excel use in business, nonprofits, and personal projects

*Hands-On Activity:* Work in pairs to clean and summarize a small dataset (e.g., women's cooperative savings)
*Mini-Assignment:* Submit a cleaned dataset with at least two basic insights (totals and averages)

---

**Session 8: Capstone Project — Everyday Data Story**

Each participant analyzes a real or provided dataset and creates a **one-page Excel report** that includes:

- A cleaned and formatted dataset
- Three or more formulas or functions
- One chart showing key insights
- A short written data story (3–5 sentences) describing the findings

**Presentation format:** Participants share their project in small breakout groups for peer feedback and facilitator review.
