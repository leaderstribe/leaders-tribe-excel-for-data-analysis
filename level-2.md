# Level 2 — Intermediate: From Data to Insight

---

## Course Overview

| Detail | Information |
|---|---|
| **Level** | Intermediate |
| **Duration** | 4 weeks |
| **Sessions** | 2 sessions per week × 1.5 hours per session |
| **Mode** | Live virtual classes with practice exercises and optional Q&A |
| **Prerequisites** | Completion of Level 1 and the Train-the-Trainer requirement |

**Goal:** Equip participants with the skills to clean, analyze, and visualize real-world datasets using Excel's intermediate functions and tools.

---

## Learning Objectives

By the end of Level 2, participants will be able to:

- Use Excel's data cleaning tools effectively and consistently
- Apply lookup and conditional formulas to connect and analyze data across tables
- Build and interpret PivotTables and PivotCharts
- Utilize data validation and dropdowns to improve data entry accuracy
- Tell clear data stories through dynamic dashboards and visual summaries

---

## Weekly Breakdown

---

### Week 1 — Strengthening Data Foundations
**Theme:** *Clean Data, Clear Insights*

**Session 1: Data Cleaning Tools and Techniques**
- Using Remove Duplicates, Text-to-Columns, and Flash Fill
- Applying Find & Replace for data correction
- Handling missing values and inconsistent formats
- Standardizing text entries and date formats

*Hands-On Activity:* Clean a dataset of NGO community beneficiaries (names, regions, gender, income)
*Mini-Assignment:* Submit a properly formatted, standardized dataset ready for analysis

---

**Session 2: Data Validation and Dropdowns**
- Setting up data validation rules
- Creating dropdown lists for categories (e.g., Region, Status, Program Type)
- Applying conditional formatting to validated data
- Preventing common data entry errors

*Hands-On Activity:* Build a validated Excel sheet for an NGO field survey with dropdowns for Gender, State, and Program Type
*Mini-Assignment:* Submit a clean, restricted-entry sheet that prevents data errors

---

### Week 2 — Lookup and Conditional Logic
**Theme:** *Smart Formulas for Smarter Analysis*

**Session 3: Lookup Functions — VLOOKUP, XLOOKUP, INDEX-MATCH**
- Merging and referencing data from multiple sheets and tables
- Understanding lookup ranges and references
- Using XLOOKUP for more flexible matching
- Combining INDEX and MATCH for advanced flexibility

*Hands-On Activity:* Merge two datasets — one with student names and one with scores — to create a complete grade sheet
*Mini-Assignment:* Submit a single merged table with accurate lookup results

---

**Session 4: Conditional Functions for Analysis**
- Using COUNTIF, SUMIF, and AVERAGEIF
- Handling multiple conditions with IFS and IFERROR
- Nesting formulas for deeper analysis

*Hands-On Activity:* Analyze a small business sales dataset to find total sales by product type, average sales by region, and the number of repeat customers
*Mini-Assignment:* Submit a conditional summary table showing insights by product and location

---

### Week 3 — Summarizing and Visualizing Data
**Theme:** *From Rows to Stories*

**Session 5: PivotTables and PivotCharts**
- Creating and customizing PivotTables
- Adding filters and slicers for dynamic analysis
- Generating PivotCharts for quick visual summaries
- Reading and interpreting PivotTable outputs

*Hands-On Activity:* Summarize NGO participant data by gender, region, and program outcome
*Mini-Assignment:* Submit a PivotTable report and a matching PivotChart visualizing participation rates

---

**Session 6: Dynamic Dashboards and Interactive Reports**
- Using slicers and timelines for interactivity
- Combining charts and summaries into a dashboard layout
- Linking charts to filters dynamically
- Best practices for clear, uncluttered dashboards

*Hands-On Activity:* Design a one-page interactive dashboard summarizing field data using PivotTables and conditional visuals
*Mini-Assignment:* Submit a functional dashboard summarizing program results

---

### Week 4 — Real-World Analysis and Capstone Project
**Theme:** *Insight to Action*

**Session 7: Real-World Data Analysis Practice**
- Integration of all intermediate skills: cleaning, lookup, analysis, and visualization
- Analysis of real-world datasets (NGO performance, location data, gender distribution)
- Building a full summary report with PivotTables and lookup-driven calculations

*Hands-On Activity:* Build a summary report with PivotTables and lookup-driven calculations
*Mini-Assignment:* Submit a full analysis file with charts and written insight notes

---

**Session 8: Capstone Project — Community Data in Action**

Each participant selects or receives a real dataset and must:

- Clean and validate the data
- Apply lookup and conditional formulas
- Build a PivotTable and PivotChart
- Create a dashboard summarizing key insights
- Write a 5-sentence data story interpreting the results

**Output:** A one-page Excel dashboard plus a short written insight summary
