## Suggested Condensed TTT Structure (2–4 Sessions)

If you are delivering a shortened version, here is a suggested structure:

| Session | Topics to Cover | Suggested Duration |
|---|---|---|
| Session 1 | Excel navigation, data entry, formatting, sorting & filtering | 60–90 mins |
| Session 2 | SUM, AVERAGE, MIN, MAX formulas; IF statements | 60–90 mins |
| Session 3 | Charts, conditional formatting, data summary | 60–90 mins |
| Session 4 (Optional) | Hands-on practice, group exercise, Q&A | 60 mins |
