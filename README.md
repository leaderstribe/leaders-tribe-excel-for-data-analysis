# Excel for Data Analysis for Women
### A Leaders Tribe Training Program

---

> **Empowering women with practical data skills — one spreadsheet at a time.**

This repository contains the full curriculum, datasets, facilitator resources, assessments, and tools for the **Excel for Data Analysis for Women** program — a structured, community-driven training initiative by [Leaders Tribe](https://leaderstribe.base44.app/).

---

## 🎯 Who This Program Is For

This program is designed for **women aged 18 and above**, including:

- Students and recent graduates
- Early-career professionals
- Entrepreneurs and small business owners
- NGO and community development staff
- Community advocates and peer educators

No prior Excel experience is required for Level 1. Level 2 builds directly on Level 1 skills.

## 🏗️ Program Structure

The program is delivered across **two progressive levels** over eight weeks per cohort (two sessions per week, 1.5 hours each).

### Level 1 — Beginner: Building Confidence with Excel (4 Weeks)
> *Theme: From Zero to Familiar*

Participants learn:
- Excel navigation, formatting, and file management
- Data entry, sorting, filtering, and basic cleaning
- Core formulas: SUM, AVERAGE, MIN, MAX
- IF statements and logical thinking
- Charts and basic data visualization
- Data summarization and conditional formatting

**Capstone:** *Everyday Data Story* — a one-page Excel report using real data

---

### Train-the-Trainer (TTT) — Bridge Between Levels

Before advancing to Level 2 or receiving the Level 1 Certificate, participants must:

- Redeliver the Level 1 training to **at least 10 new learners** in their community or professional network
- Ensure **at least 50% of their trainees are women**
- Submit a **TTT Reflection Report** documenting sessions, learners, photos, and takeaways

This model ensures that every trained woman becomes a multiplier of knowledge and skills.

---

### Level 2 — Intermediate: From Data to Insight (4 Weeks)
> *Theme: Clean Data, Clear Insights → Insight to Action*

Participants learn:
- Advanced data cleaning: Remove Duplicates, Text-to-Columns, Flash Fill
- Data validation and dropdown lists
- Lookup functions: VLOOKUP, XLOOKUP, INDEX-MATCH
- Conditional functions: COUNTIF, SUMIF, AVERAGEIF, IFS, IFERROR
- PivotTables and PivotCharts
- Interactive dashboards with slicers and timelines

**Capstone:** *Community Data in Action* — an interactive Excel dashboard

## 📊 About the Datasets

All practice datasets in this repository are **dummy datasets created for training purposes**. They are not real participant data.

Datasets are designed to:
- Reflect **Nigerian contexts** — names, cities, currencies, and everyday scenarios
- Support specific Excel skills being taught in each session
- Include both **messy** and **cleaned** versions where relevant for cleaning practice
- Be beginner-friendly and internally consistent enough to analyze

> All CSV datasets can be downloaded from the `datasets/` folder and opened directly in Microsoft Excel.
