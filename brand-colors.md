## Color Gradient (Logo Feather)
The logo feather uses a gradient: `#F4811F` (tip) → `#D94F3D` (mid) → `#7B2D8B` (base/quill)

This gradient can be used for:
- Button hover effects
- Section divider lines
- Certificate header bars
- Slide deck title backgrounds

CSS gradient: `background: linear-gradient(135deg, #F4811F 0%, #D94F3D 50%, #7B2D8B 100%);`
